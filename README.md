# Enrollment-Management-System
基于 Spring Boot + Vue 的招生管理系统，是一款为招生工作服务的平台。系统支持招生信息发布、学生报名、信息筛选、录取流程管理等功能。前端 Vue 提供清晰直观的操作界面，方便招生人员和考生操作；后端 Spring Boot 确保系统稳定，保障招生信息和用户数据的安全存储和高效处理，能有效提高招生工作的效率和管理水平。
